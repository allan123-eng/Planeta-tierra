Sistema web Geográfico

Aplicación web interactiva diseñada para explorar y aprender sobre la geografía de Nicaragua. Los usuarios pueden navegar por un mapa dividido en departamentos y zonas de interés, acceder a información detallada sobre cada región, resolver cuestionarios y puzzles geográficos, y registrar su progreso. La plataforma incluye funcionalidades personalizadas para estudiantes, maestros y administradores, ofreciendo herramientas educativas innovadoras y una experiencia dinámica para fomentar el conocimiento geográfico.
